# Deployment Checklist
*Perform these checks after pushing to GitHub.*

1.  **Settings**: Is GitHub Pages enabled in Settings > Pages?
2.  **Source**: Is it set to `main` branch (and root folder)?
3.  **Public**: Is the repository visibility set to Public?
4.  **Homepage**: Is the main file named exactly `index.html` (lowercase)?
5.  **Links**: Click every nav link on the live site. Do they work?
6.  **Console**: Press F12. Are there any red errors about missing files?
