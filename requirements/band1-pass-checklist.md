# Band 1 Pass Checklist
*Use this to check off items before submitting.*

- [ ] **Semantic HTML**: Do I use <header>, <nav>, <main>, <footer>?
- [ ] **3 Pages**: Do I have index.html, about.html, portfolio.html?
- [ ] **Navigation**: Does the menu work on ALL pages?
- [ ] **Responsive**: Does it look good on mobile (no horizontal scroll)?
- [ ] **Bootstrap**: Am I using Bootstrap classes (container, row, col)?
- [ ] **Portfolio**: Are my project items consistent (same HTML structure)?
- [ ] **Images**: Do all images load? (Check capitalization! Image.jpg vs image.jpg)
- [ ] **GitHub**: Do I have multiple commits? (Not just one "upload")
